# 🍅 番茄钟

一个零依赖、单文件的网页番茄钟。双击 `index.html` 即可在浏览器里运行。

## 功能

- 经典 25/5 番茄工作法,每完成 4 个工作番茄自动进入 15 分钟长休
- 任务列表:添加、勾选完成、删除,选中的任务在工作番茄结束后自动 +1
- 完成时三连铃声 + 桌面通知(首次使用需在浏览器允许通知权限)
- 任务和当前选中状态保存在 `localStorage`,刷新不丢失
- 空格键快捷开始 / 暂停

## 使用

1. 双击 `index.html`,在浏览器中打开
2. 首次开始时浏览器会询问"是否允许通知",点允许
3. 添加任务 → 点击任务名选中 → 按"开始"
